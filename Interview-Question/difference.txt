what is the difference betweeen library, package and Modules

| Term        | What it is                             | Example                         | Analogy                            |
| ----------- | -------------------------------------- | ------------------------------- | ---------------------------------- |
| **Module**  | A single `.py` file with code          | `math.py`, `random.py`          | A single book                      |
| **Package** | A folder of modules with `__init__.py` | `numpy/core/`                   | A bookshelf with related books     |
| **Library** | A collection of packages/modules       | `NumPy`, `Pandas`, `Matplotlib` | A library building full of shelves |
